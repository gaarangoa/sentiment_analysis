## Sentiment analysis of spanish sentences
This repository is built to predict the sentiment of sentences written in spanish. It has been trained with open source datasets (wiki and twitter). 

### Requirements
    Tensorflow 1.0.1
    python 2.7
    tflearn 0.3
    flask 0.12.2
    numpy 1.12.1
    scipy 0.19.0
